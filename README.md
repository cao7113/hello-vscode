## 调试插件

- 在 VS Code 中打开项目。
- 按 F5 启动调试，VS Code 会打开一个新的扩展开发主机窗口。
- 在命令面板中（Ctrl+Shift+P 或 Cmd+Shift+P），运行 Hello World 命令，查看结果。

### 注意

- 在 src/extension.ts 时 按 `Fn + F5` 进入调试开发机
- tsconfig.json中指定 使用 CommonJS, 本质上是nodejs应用
- xxx.vsix文件右键直接可安装
